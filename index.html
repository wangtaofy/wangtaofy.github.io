HEXO 博客安装与部署

HEXO 博客安装与部署

首先需要安装 node、npm，建议用 nvm 来安装 node

安装完成后执行下列命令安装 hexo

$ npm install hexo-cli -g

在本地新建存放博客文件的文件夹：hexo文件夹，文件夹的路径中尽量不要有中文。

进入该文件夹，右键，Git Bash，执行以下命令，HEXO 会自动下载搭建网站所需的所有文件

$ hexo init
