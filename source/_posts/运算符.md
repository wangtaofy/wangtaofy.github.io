# 运算符

js 拥有`一元运算符`和`二元运算符`，以及一个特殊的`三元运算符(即条件运算符)`。

一元运算符需要一个操作数和一个运算符

```
	+a -a ++a --a a++ a--
```

二元运算符需要两个操作数和一个操作符

```
	1+2  3*4
```

## 赋值运算符

将右边操作数的值赋给左边的操作数，一个最简单的赋值运算符是 '='。

x = y，将 y 的值赋给 x。

### 复合赋值运算符

|   名字		           |   简写		       |   含义             |    
| ---------		         | --------:       | :-------:          |
|   加法赋值           |   x += y        |   x = x + y        |
|   减法赋值           |   x -= y        |   x = x - y        |
|   乘法赋值           |   x \*= y       |   x = x * y        |
|   除法赋值           |   x /= y        |   x = x / y        |
|   求余赋值           |   x %= y        |   x = x % y        |
|   求幂赋值           |   x **= y       |   x = x ** y       |
|   左移位赋值         |   x <<= y       |   x = x << y       |
|   右移位赋值         |   x >>= y       |   x = x >> y       |
|   无符号右移位赋值   |   x >>>= y      |   x = x >>> y      |
|   按位与赋值         |   x &= y        |   x = x & y        |
|   按位或赋值         |   x &#124;= y   |   x = x &#124; y   |
|   按位异或赋值       |   x ^= y        |   x = x ^ y        |

### 解构

结构语法是一个能从数组或对象的数组结构或对象字面量中提取数据的 js 表达式。

- 解构数组

```
	var foo = ['one', 'two', 'three'];

	var [one, two, three] = foo;
	console.log(one);    // one
	console.log(two);		 // two
	console.log(three);	 // three
```

你可以对任意深度的数组解构。

```
	var [foo, [bar, [baz]]] = [1, [2, [3]]];
	console.log(foo, bar, baz);  // 1 2 3
```

可以在对应位留空来跳过解构的数组中的元素。

```
	var [ , , baz] = [1, 2, 3];
	console.log(baz);  // 3
```

还可以通过`不定参数`模式来捕获数组中的尾随元素。

```
	var [foo, ...tail] = [1, 2, 3, 4, 5];
	console.log(tail);  // [2, 3, 4, 5]
```

- 解构对象

```
	var foo = {
	    name: 'hoho',
		age: 18,
		sex: '男'
	};

	var {name: one, age: two, sex: three} = foo;
	console.log(one);    // hoho
	console.log(two);		 // 18
	console.log(three);	 // 男
```

当要解构的属性名与要赋值的变量名一致时，可以用一种简写的方式。

```
	var obj = {
		foo: '645'
	}

	var {foo} = obj;
```

跟解构数组一样，你可以随意嵌套进一步组合解构。

```
	var obj = {
		foo: ['897']
	}

	var {foo: [num]} = obj;
	console.log(foo, num);  // foo 897
```

当解构一个未定义的属性时，会返回 undefined。

```
	var {foo} = {};
	console.log(foo);  // undefined
```

如果你解构属性后赋值的变量已经声明，你必须用括号将整个表达式括起来，否则会返回一个错误。

```
	var foo = 5;
	{foo} = {foo: 10};
	console.log(foo);  // Uncaught SyntaxError: Unexpected token =
```

```
	var foo = 5;
	({foo} = {foo: 10});
	console.log(foo);  // 10
```
