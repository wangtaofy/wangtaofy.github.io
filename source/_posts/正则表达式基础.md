# 语法

> let reg = /pattern(模式)/attribute(修饰符);

# 字符串方法

* match：返回一个包含匹配项的数组

> str.match(reg)

* replace：替换字符串中的匹配项

> str.replace(reg, 要替换成的值)

<!--more-->

# 正则表达式方法

* test：判断字符串中是否含有匹配项，有，返回 true，否则，返回 false

> reg.test(str)

* exec：查找字符串中符合正则表达式的匹配项，有返回匹配项和匹配项的位置，否则返回 null

> reg.exec(str)

# 修饰符

1、g：全局匹配（正则表达式匹配时默认匹配到第一项时就停止）

2、i：匹配时不区分大小写

3、m：多行匹配

# []：匹配某个范围内的单个字符

* [abc]：匹配方括号中的任意一个字符

* [^abc]：匹配任何不在方括号中的字符

* [a-z]：匹配a-z之间的小写字符

* [A-Z]：匹配A-Z之间的大写字符

* [0-9]：匹配0-9之间的数字字符

# 元字符

* .：匹配除了换行符和行结束符之外的所有单个字符

* \w：匹配数字、字母、下划线，等价于 [a-zA-Z0-9_]

* \W：匹配数字、字母、下划线之外的字符

* \s：匹配空白字符

* \S：匹配非空白字符

* \d：匹配数字

* \D：匹配非数字

* \b：匹配单词边界，即单词的上一个和下一个分隔符

* \B：匹配非单词边界，即匹配项的上一个和下一个字符必须都是单词或非单词字符

# 量词

* +：表示前面的字符出现一次或多次

* *：表示前面的字符出现零次或多次

* ?：表示前面的字符出现了0或1次

* {n}：表示前面的字符出现了n次

* {n,}：表示前面的字符至少出现了n次

* {n,m}：表示前面的字符至少出现n次，至多出现m次

* ^n：匹配以某个字符开头的字符串

* n$：匹配以某个字符结尾的字符串
